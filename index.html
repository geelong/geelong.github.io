#列表
列表格式也很常用，在 Markdown 中，你只需要在文字前面加上 - 就可以了，例如：

- a

- b

- c

  • a

  • b

  • c

有序列表

\1. a

\1. b

\1. c

  1. a

  2. b

  3. c

#链接和图片
在 Markdown 中，插入链接不需要其他按钮，你只需要使用

链接

[显示文本](链接地址)

显示文本

图片

!\

abc

#粗体和斜体
用两个 包含一段文本就是粗体的语法，用一个 包含一段文本就是斜体的语法。例如：

*一盏灯*

一盏灯

**一简书**

一简书

前言

  1. 编译一个userdebug环境的好处是可以debug各种源码，包括system process等，比光看源码要容易一些。

  2. 开发的时候使用也比较方便，在终端直接看APP的数据，使用 adb shell ls /data/data/包名/files 等

  3. 之前是想用模拟器的，不过我们的项目只有arm的so库，模拟器兼容性不是很好

过程

  1. 下载大概是24小时（视网络情况定）
  2. 编译大概24小时
