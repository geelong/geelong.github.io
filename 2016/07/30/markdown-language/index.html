markdown 语法

发表于 2016-07-30   |     |   阅读次数

前言

例子中先显示语法，再显示效果

参考文章：Markdown 语法入门（Mou）

标题

只需要在文本前面加上 # 即可

# 一级标题

## 二级标题

### 三级标题

#### 四级标题

##### 五级标题

###### 六级标题

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

列表

列表格式也很常用，在 Markdown 中，你只需要在文字前面加上 - 就可以了，例如：

- a

- b

- c

  • a

  • b

  • c

有序列表

\1. a

\1. b

\1. c

  1. a

  2. b

  3. c

链接和图片

在 Markdown 中，插入链接不需要其他按钮，你只需要使用

链接

[显示文本](链接地址)

显示文本

图片

!\

引用

> abc

abc

粗体和斜体

用两个 包含一段文本就是粗体的语法，用一个 包含一段文本就是斜体的语法。例如：

*一盏灯*

一盏灯

**一简书**

一简书
