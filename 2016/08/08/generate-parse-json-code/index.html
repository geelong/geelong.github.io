下面的工具可以根据json的数据，自动生成json的解析代码

使用步骤

1. GsonFormat安装

  • AndroidStudio-preferences-plugins-install plugin from disk

  • 选择plugin_gsonformat.jar文件

  • 根据提示，重启AndroidStudio

2.生成Json解析代码

alt+s 弹出生成代码的对话框(快捷键可能不同，可以在快捷键列表里查一下)

在对话框里张贴json格式的数据

一直点击OK
